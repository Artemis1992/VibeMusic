| Метод              | Описание                                                                 | Основные параметры                                                                 | Пример использования                                                                 | Возвращаемый объект                     | Возможные исключения                              | Примечания                                                                 |
|:-------------------|:-------------------------------------------------------------------------|:----------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------|:---------------------------------------|:------------------------------------------------|:---------------------------------------------------------------------------|
| `requests.get()`   | Выполняет HTTP GET-запрос для получения данных с сервера                 | `url`, `params` (словарь для query-параметров), `headers`, `timeout`, `auth`, `cookies` | ```python<br>import requests<br>response = requests.get('https://api.example.com/data', params={'id': 1}, headers={'Authorization': 'Bearer token'}, timeout=5)<br>print(response.json())<br>``` | `Response` (содержит `status_code`, `text`, `json()`, `content`) | `RequestException`, `HTTPError`, `ConnectionError`, `Timeout` | Используется для получения ресурсов, поддерживает query-параметры (`?key=value`) |
| `requests.post()`  | Выполняет HTTP POST-запрос для отправки данных на сервер                 | `url`, `data` (словарь или строка), `json` (для JSON-данных), `files`, `headers`, `timeout`, `auth` | ```python<br>import requests<br>payload = {'key': 'value'}<br>response = requests.post('https://api.example.com/submit', json=payload, headers={'Content-Type': 'application/json'})<br>print(response.status_code)<br>``` | `Response` | `RequestException`, `HTTPError`, `ConnectionError`, `Timeout` | Подходит для отправки форм, JSON или файлов (multipart/form-data)           |
| `requests.put()`   | Выполняет HTTP PUT-запрос для полного обновления ресурса на сервере      | `url`, `data`, `json`, `headers`, `timeout`, `auth`, `cookies`                     | ```python<br>import requests<br>response = requests.put('https://api.example.com/id/1', json={'key': 'updated_value'}, headers={'Authorization': 'Bearer token'})<br>print(response.json())<br>``` | `Response` | `RequestException`, `HTTPError`, `ConnectionError`, `Timeout` | Заменяет весь ресурс новыми данными                                        |
| `requests.delete()`| Выполняет HTTP DELETE-запрос для удаления ресурса на сервере             | `url`, `headers`, `timeout`, `auth`, `cookies`                                     | ```python<br>import requests<br>response = requests.delete('https://api.example.com/id/1', headers={'Authorization': 'Bearer token'})<br>print(response.status_code)<br>``` | `Response` | `RequestException`, `HTTPError`, `ConnectionError`, `Timeout` | Удаляет указанный ресурс, ответ часто пустой (204 No Content)              |
| `requests.head()`  | Выполняет HTTP HEAD-запрос для получения заголовков ответа без тела      | `url`, `headers`, `timeout`, `auth`, `cookies`                                     | ```python<br>import requests<br>response = requests.head('https://api.example.com')<br>print(response.headers)<br>``` | `Response` (только заголовки) | `RequestException`, `HTTPError`, `ConnectionError`, `Timeout` | Полезен для проверки метаданных (например, Content-Length, Last-Modified)   |
| `requests.options()`| Выполняет HTTP OPTIONS-запрос для получения поддерживаемых методов сервера | `url`, `headers`, `timeout`, `auth`, `cookies`                                     | ```python<br>import requests<br>response = requests.options('https://api.example.com')<br>print(response.headers['Allow'])<br>``` | `Response` | `RequestException`, `HTTPError`, `ConnectionError`, `Timeout` | Возвращает методы, поддерживаемые сервером (например, GET, POST)           |
| `requests.patch()` | Выполняет HTTP PATCH-запрос для частичного обновления ресурса на сервере | `url`, `data`, `json`, `headers`, `timeout`, `auth`, `cookies`                     | ```python<br>import requests<br>response = requests.patch('https://api.example.com/id/1', json={'key': 'new_value'}, headers={'Authorization': 'Bearer token'})<br>print(response.json())<br>``` | `Response` | `RequestException`, `HTTPError`, `ConnectionError`, `Timeout` | Обновляет только указанные поля ресурса                                    |
